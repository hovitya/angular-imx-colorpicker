imxColorShades
directive in module imx.colorPicker

Description

Render hue, brightness and saturation input control.

Dependencies

  • imxPaletteService
  • imxResize

Usage

as element:
<imx-color-shades
       ng-model="{string}">
</imx-color-shades>
as attribute
<ANY imx-color-shades
     ng-model="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
ngModelstring

Assignable angular expression to data-bind to. Data will be bind in hex format.
