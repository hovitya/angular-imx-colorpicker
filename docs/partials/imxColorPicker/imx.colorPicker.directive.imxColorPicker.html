imxColorPicker
directive in module imx.colorPicker

Description

Render color picker component.

Example
<imx-color-picker format="hex" ng-model="myColor"></imx-color-picker>

Dependencies

  • imxPaletteService
  • imxColorStoreService

Usage

as element:
<imx-color-picker
       template-url="{string}"
       format="{string}"
       closable="{boolean}"
       base-colors="{=Array}"
       ng-model="{string}"
       on-close="{function}">
</imx-color-picker>
as attribute
<ANY imx-color-picker
     template-url="{string}"
     format="{string}"
     closable="{boolean}"
     base-colors="{=Array}"
     ng-model="{string}"
     on-close="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
templateUrlstring

Color picker html template url.

formatstring

Output color format (hex|hsl|rgb)

closableboolean

Display close button on color picker ui and call provided on-close function when it clicked. Current selected color will be added to last used list on close.

baseColors=Array

Solid colors will be populated using these base colors. (array of color string)

ngModelstring

Assignable angular expression to data-bind to. Data will be bind in the specified format.

onClosefunction

Close callback.
