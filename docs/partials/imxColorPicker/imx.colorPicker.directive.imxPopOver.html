imxPopOver
directive in module imx.colorPicker

Usage

as element:
<imx-pop-over
       for="{string}"
       trigger="{string}">
</imx-pop-over>
as attribute
<ANY imx-pop-over
     for="{string}"
     trigger="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
forstring

Selector for host component. e.g. #buttonId

triggerstring

Trigger type. The default trigger type is click. Click means popup opened when user clicks on host element and it will be dismissed when clicks again. If value set to hover mouse over event will trigger this popover and mouse out will dismiss it. If you want to control popover manually set trigger to none.
