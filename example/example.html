IMX ColorPicker component Examples

Using as inline element

You can use color picker component as an embedded element. Simply add imx-color-picker tag to your code.

HTML code

<imx-color-picker selected-color="color"></imx-color-picker>

Javascript code

angular.module('demo',['imx.colorPicker'])
    .controller('demoCtrl',  ['$scope', function($scope) {
    $scope.color = '#000FFF';
}]);

Using as color input

The module defines a new input type called imx-color. As result of using this type it replaces default input behaviour with color input. It is compatible with the ngModel directive. Click on the input field to test it.

HTML code

<imx-color-picker selected-color="color"></imx-color-picker>

Javascript code

angular.module('demo',['imx.colorPicker'])
    .controller('demoCtrl',  ['$scope', function($scope) {
    $scope.color = '#000FFF';
}]);
